API 使用说明

端点地址

POST /api/v1/chat/completions

请求示例

curl -X POST http://localhost:8080/api/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{
    "model": "minimax/minimax-m1",
    "messages": [
      {
        "role": "user",
        "content": "Hello, how are you?"
      }
    ],
    "stream": false
  }'

流式请求示例 

curl -X POST http://localhost:8080/api/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{
    "model": "minimax/minimax-m1",
    "messages": [
      {
        "role": "user",
        "content": "Tell me a story"
      }
    ],
    "stream": true
  }'

支持的参数

参数 类型 必需 说明
model string 是 模型名称，如 minimax/minimax-m1
messages array 是 对话消息数组
stream boolean 否 是否启用流式响应
temperature number 否 温度参数 (0-2)
max_tokens integer 否 最大生成token数

Gemini 原生接口端点

POST /v1beta/models/{model}:generateContent - 非流式生成
POST /v1beta/models/{model}:streamGenerateContent - 流式生成
GET /v1beta/models - 获取模型列表

非流式请求示例 

curl -X POST http://localhost:8080/v1beta/models/gemini-2.5-pro:generateContent \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "contents": [
      {
        "role": "user",
        "parts": [
          {
            "text": "请解释一下量子计算的基本原理"
          }
        ]
      }
    ],
    "generationConfig": {
      "temperature": 0.7,
      "maxOutputTokens": 1000,
      "thinkingConfig": {
        "includeThoughts": true
      }
    }
  }'

流式请求示例 

curl -X POST http://localhost:8080/v1beta/models/gemini-2.5-pro:streamGenerateContent \
  -H "Content-Type: application/json" \
  -H "x-api-key: YOUR_API_KEY" \
  -d '{
    "contents": [
      {
        "role": "user",
        "parts": [
          {
            "text": "写一个关于人工智能的故事"
          }
        ]
      }
    ],
    "generationConfig": {
      "temperature": 0.8,
      "maxOutputTokens": 2000,
      "thinkingConfig": {
        "includeThoughts": true
      }
    }
  }'

支持的模型

  • gemini-2.5-pro - Gemini 2.5 Pro (支持思考模式)
  • gemini-2.5-flash - Gemini 2.5 Flash (支持思考模式)
  • gemini-1.5-pro - Gemini 1.5 Pro
  • gemini-1.5-flash - Gemini 1.5 Flash

Gemini 原生参数说明

参数 类型 必需 说明
contents array 是 对话内容数组，包含role和parts
generationConfig object 否 生成配置参数
temperature number 否 温度参数 (0-2)
maxOutputTokens integer 否 最大输出token数
thinkingConfig object 否 思考模式配置 (仅2.5系列)
includeThoughts boolean 否 是否包含思考内容在响应中

思考模式响应示例 

{
  "candidates": [
    {
      "content": {
        "parts": [

          {
            "text": "量子计算是基于量子力学原理的计算方式..."
          }
        ],
        "role": "model"
      },
      "index": 0
    }
  ],
  "usageMetadata": {
    "promptTokenCount": 18,
    "candidatesTokenCount": 150,
    "totalTokenCount": 168,
    "thoughtsTokenCount": 50
  }
}

特殊功能

思考模式 (Thinking Mode)

Gemini 2.5 系列模型支持思考模式，可以在响应中包含模型的思考过程：

  • 设置 thinkingConfig.includeThoughts: true 启用
  • 响应中包含 "thought": true 的部分为思考内容
  • 使用统计中包含 thoughtsTokenCount 字段
  • 思考内容保持原汁原味，不做任何处理
